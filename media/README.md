# Media Files

Large media files not tracked in git. Download manually:

## Perseverance Landing Video (2160p)

Official NASA onboard camera footage of Perseverance's descent and touchdown on Mars.

**Download:**
```bash
curl -o media/Perseverance-landing-2160p.mp4 \
  "https://assets.science.nasa.gov/content/dam/science/psd/mars/downloadable_items/45703_JPL-20210222-M2020f-0001-Perseverance_Rovers_Descent_and_Touchdown_on_Mars-1.mp4"
```

**Source:** [NASA Mars - Perseverance Descent and Touchdown](https://mars.nasa.gov/resources/25628/perseverance-rovers-descent-and-touchdown-on-mars-onboard-camera-views/)
